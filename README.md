# 餐厅收银系统简介 python243

## 项目概述

本项目是基于 Python 3.10 和 Django 4.2 开发的餐厅收银系统。系统采用前后端分离的技术架构，前端使用 Ajax 技术实现数据的异步交互。

## 技术栈

- Python 3.10
- Django 4.2
- Ajax

## 功能模块

### 顾客端

- 点餐：实现基础点餐操作。
- 订单状态查询：实时查看订单处理状态。
- 菜品明细查询：详细展示菜品信息。

### 管理员端

#### 功能模块

- 点餐管理
- 后厨管理：实现上菜、退单等功能。
- 订单管理：查看订单明细。
- 库存流水管理：监控库存变动情况。
- 财务报表：支持多时间维度查看销售数据。
- 信息管理页面

#### 信息管理

- 菜品数据管理：支持增删查改。
- 员工数据管理：支持增删查改。
- 桌号数据管理：支持增删查改。

## 系统角色

- 顾客
- 管理员
